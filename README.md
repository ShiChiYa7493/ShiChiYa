# shichiya-vue

> An Online Library By ShiChiYa

## Build Setup

``` bash
# SpringBoot+Vue前后端分离式框架构建的图书馆的模仿作品
# 根目录下sql文件用于生成数据库（开发使用的是MySQL 8.0.12）
# 使用IDEA分别打开前端(shichiya-vue)和后端(shichiya)
# 后端打开后右下角提示安装依赖选择安装，如果未弹出提示安装
可右键pom.xml选择Maven->Reimport进行安装
# 前端同样需要安装依赖包,于Terminal中在shichiya根目录输入
npm install
# 依赖安装完成后，启动前端与后端。前端于Terminal中输入
npm run dev
后端直接运行src/main/java/com.example.shichiya/ShiChiYaApplication.java
# 项目运行与http://localhost:8080/index  初始账号admin 密码123
`

该项目为模仿的毕业设计，原作者教程https://learner.blog.csdn.net/article/details/88925013
